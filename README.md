# Sabairecker POS

Este repositorio contiene un ejemplo simple de un sistema de punto de venta (POS) para un restaurante implementado en Python.

## Requisitos

- Python 3.8 o superior.

## Uso

1. Clonar el repositorio.
2. Ejecutar el script:

```bash
python src/pos.py
```

El programa mostrará un menú de productos, permitirá seleccionar artículos, calculará el total y procesará el pago.

## Características

- Menú básico con cuatro productos.
- Selección de artículos mediante números.
- Cálculo automático del total y del cambio.

Este proyecto es un ejemplo sencillo para fines educativos y puede expandirse según las necesidades del restaurante.
